EventSphere

"A comprehensive Event Management and Room Booking System for Academic Institutions providing end-to-end solutions for campus event coordination with secure authentication and role-based access control."

Problem Statement

Academic institutions lack centralized systems for event management and room allocation, resulting in scheduling conflicts, poor coordination between administrators and users, duplicate bookings, and inefficient workflows. Current manual processes lead to operational inefficiencies and communication gaps across campus.

Existing Solutions

Manual room allocation, disconnected event management systems, spreadsheet-based tracking, limited user role management, no automated conflict detection.

Our Innovation

Slot-based architecture for multi-day events, per-day room allocation with conflict detection, JWT-secured RBAC, automated approval workflows, real-time notifications, and concurrent-safe booking operations.

System Architecture

Backend Stack

Spring Boot 3.x, Spring Security (JWT), Spring Data JPA, PostgreSQL, Pessimistic Locking for concurrency

Frontend Stack

React 18 + Vite, Axios, JWT Token Management, Calendar UI, Responsive Forms, Role-Based Navigation

Fully functional full-stack system with slot-based booking, conflict-free scheduling, role-based access control, JWT authentication, and automated room allocation workflows deployed on Render (backend) and Vercel (frontend).

Multi-Day Event Handling

Automatic slot generation for single-day, multi-day, and overnight events. Per-day room allocation ensures flexibility in venue assignment across event duration.

Validation Engine

Real-time validation against building hours, fixed timetables, and existing bookings. Concurrent operations protected by pessimistic locking mechanisms.

Notification System

Real-time alerts for booking confirmations, role approvals, and room allocations. Seamless communication between users and administrators across roles.
